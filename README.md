# legacy-crm-etl-pipeline

## Overview
This project simulates a real-world data engineering workflow by migrating a legacy CRM system from a SQL Server backup into a modern analytics-ready PostgreSQL warehouse.

The goal is to demonstrate end-to-end skills in:
- Data extraction from legacy systems (.bak SQL Server backup)
- Schema reverse engineering
- ETL pipeline development in Python
- Data validation and quality checks
- Relational data modeling for analytics

## Tech Stack
- SQL Server (source system)
- PostgreSQL (target warehouse)
- Python (ETL pipeline)
- SQLAlchemy
- pandas
- Git & GitHub
